# videoCheck
Check video files for missing audio or video

[![Build](https://github.com/jcalado/videoCheck/actions/workflows/build.yml/badge.svg)](https://github.com/jcalado/videoCheck/actions/workflows/build.yml)
[![Release](https://github.com/jcalado/videoCheck/actions/workflows/release.yml/badge.svg)](https://github.com/jcalado/videoCheck/actions/workflows/release.yml)

## How to

Either:  
- Drop a file over the window  
- Click the button, pick a file

## Requirements
### ffmpeg
- You can install via [Chocolatey](https://chocolatey.org/) on windows, your distro's package manager on linux, and [homebrew](https://brew.sh/) on macOS.

or

- Install via ffmpeg's website: https://www.ffmpeg.org/download.html
